1. Introduction

RSG Consulting is committed to protecting your privacy. Zenicca (CEO Advisor) is an AI-driven platform that provides real-time strategic guidance, wellness insights, and productivity recommendations for CEOs.

Zenicca integrates data from Garmin Health API, Microsoft Graph, Hogan Assessments, and Geopolitical APIs to deliver highly personalized coaching and recommendations. We do not sell, share, or monetize user data. All data is used strictly for improving executive performance and well-being.

2. Data Collected & Purpose of Use

Zenicca collects and processes the following data:

Data Source Data Collected Purpose
Garmin Health API Heart rate, stress levels, sleep, activity metrics Monitor wellness, stress management, energy tracking
Microsoft Graph API Calendar events, email metadata Identify work patterns, suggest focus & productivity tips
Hogan Assessments Leadership traits, team dynamics Provide personalized leadership coaching
Geopolitical APIs Real-time economic & political trends Deliver market-aware strategic insights

3. User Consent & Data Control

  • Users must explicitly grant access before Zenicca can process data.
  • Users can revoke access at any time via Garmin Connect settings or Microsoft API permissions.
  • Users have full control over which insights they receive.

4. Data Security & Compliance

Zenicca follows strict security protocols to protect user data:

  • All data is encrypted at rest and in transit.
  • OAuth2.0 authentication for secure API connections.
  • GDPR & CCPA compliant with transparent data retention policies.

5. Third-Party Data Sharing

We DO NOT sell or share data with any third parties. User data is processed exclusively for personalized coaching and productivity enhancements within Zenicca.
